测试数据  
该数据集包含由 OpenAI 的 AI 语言模型 ChatGPT 提供的两个测试样本。  
这些样本包括一个 markdown 表格和一段英文文本，可用于测试支持文本和表格格式的英译汉翻译软件。  

文本测试  
快速的棕色狐狸跳过懒惰的狗。这个全字母句至少包含英语字母表中的每个字母一次。全字母句常用于测试字体、键盘和其他与文本相关的工具。除了英语，许多其他语言中也有全字母句。由于语言的独特特性，有些全字母句的构造更加困难。
